# 安裝手冊

## 需求

- Python 3.x

## 安裝步驟

1. 克隆此專案到本地端：
    ```sh
    git clone <repository-url>
    ```
2. 進入專案目錄：
    ```sh
    cd <repository-directory>
    ```
3. 安裝所需的套件：
    ```sh
    pip install -r requirements.txt
    ```

## 執行測試

1. 執行 `test.py` 檔案：
    ```sh
    python test.py
    ```
